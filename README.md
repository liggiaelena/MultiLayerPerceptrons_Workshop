# Multi-Layer Perceptrons Workshop

## Overview
This workshop explores the mechanics of **forward** and **backward propagation** in multi-layer neural networks. Participants gain hands-on experience implementing the learning process from scratch using **NumPy**, deepening their understanding of how deep learning models train and optimize.

## Objectives
- Understand how information flows through multiple hidden layers.
- Implement **forward propagation** to compute activations and predictions.
- Apply **backpropagation** to compute gradients and update parameters.
- Visualize the learning process and analyze training behavior.

## Activities
1. **Build a 3-layer neural network** with ReLU or sigmoid activations.
2. Implement **gradient descent** for parameter updates.
3. Experiment with different activation functions, learning rates, and layer depths.
4. Visualize loss curves and discuss the impact of network depth.
5. Reflect on issues like **vanishing/exploding gradients**.

## Deliverables
- A Jupyter Notebook implementing forward and backward propagation.
- A brief written reflection on insights gained during the workshop.
- (Optional) A simple classifier applied to a toy dataset like XOR or MNIST.

## Key Takeaways
By completing this workshop, students will:
- Develop a deeper intuition for how neural networks learn.
- Understand the role of each layer in information and gradient flow.
- Gain confidence in debugging and optimizing deep learning models.
